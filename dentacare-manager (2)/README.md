
# DentaCare Manager - Deployment Guide | دليل تشغيل البرنامج

## English Instructions (How to get your link):
1. **Download the files**: Copy all the files in this project to a folder on your computer.
2. **Easiest Way (Netlify Drop)**:
   - Go to [Netlify Drop](https://app.netlify.com/drop).
   - Drag and drop your folder into the box.
   - You will get a link immediately (e.g., `https://random-name.netlify.app`).
3. **Professional Way (Vercel/GitHub)**:
   - Upload files to a GitHub repository.
   - Connect the repository to [Vercel](https://vercel.com).
   - It will automatically deploy and give you a permanent link.

---

## التعليمات بالعربية (كيف تحصل على رابط خاص بك):

1. **تحميل الملفات**: قم بنسخ كافة ملفات هذا المشروع في مجلد واحد على جهازك.
2. **الطريقة الأسهل (Netlify Drop)**:
   - اذهب إلى موقع [Netlify Drop](https://app.netlify.com/drop).
   - قم بسحب المجلد الذي يحتوي على الملفات وإلقائه داخل المربع المخصص.
   - سيعطيك الموقع رابطاً فورياً (مثلاً: `https://clinic-name.netlify.app`).
3. **الطريقة الاحترافية (GitHub + Vercel)**:
   - ارفع الملفات على حسابك في GitHub.
   - اربط الحساب بموقع Vercel، وسيتم تحديث الرابط تلقائياً كلما عدلت الكود.

**ملاحظة**: بمجرد فتح الرابط على موبايلك، استخدم خيار "إضافة إلى الشاشة الرئيسية" ليتحول الرابط إلى تطبيق حقيقي.
